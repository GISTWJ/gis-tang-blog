2、如何让裁剪空间坐标使用屏幕像素坐标？

1. 屏幕坐标先除以画布分辨率进行归一化
2. 将归一化坐标乘以2，以第一象限代替四个象限的显示范围
3. 将第一象限向左下平移，使坐标原点位于屏幕左下角
4. 如果想要参考传统二维坐标原点在左上角，可以进行Y轴翻转，右点乘一个向量(1,-1)
